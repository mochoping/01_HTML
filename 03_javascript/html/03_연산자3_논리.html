논리 부정(NOT) 연산자 (!)

    논리값 반대로 변환

    예시
    const 사과 = "apple";

    사과 != "apple" ? "사과가 apple 이 아니라 grape 입니까? "일때 보이는곳 : "여기에 값이 보임";

부등호 (=, ==, ===)

    =  (할당연산자)
        - 변수에 값을 할당할 때 사용
        - ex) const a = 5; let 사과 = "빨간색" ;  x = 5;

    == (동등연산자)
        - 값만 비교하는 연산자. 타입이 다르더라도 값이 같으면 true 반환
        - ex)
            const 숫자5  = 5;
            const 글자5  ="5";
            숫자5 == 글자5 -> true 
            값을 우선적으로 보기때문 스크립가 덜 예민한것
            
    ===(일치연산자)
        - 값과 타입이 모두 일치하는지 확인, 
        - ex)
            const 숫자5  = 5;
            const 글자5  ="5";
            숫자5 == 글자5 -> false
            엄격한 비교를 원할때 사용

    == 는 타입변환이 자동으로 이루어지지만 일어나는 기준을 알 수 없음
    ===는 타입변환이 자동으로 이루어지지 않게 설정함
