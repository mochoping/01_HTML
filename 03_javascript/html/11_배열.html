배열 (array)

        -변수를 묶음으로 다루는것 (자료구조의 기초)
        (js 변수의 특징 : 대입되는 값에 따라서 자료형이 변함)

        -> js 배열 : 자료형 상관없이 여러값을 한번에 다룸

        - js 배열 특징 
        자료형 제한 없음(보통은 한가지 자료형만 저장)
        길이제한없음(배열 길이가 자동 늘어났다 줄어들었다가 함)

        [js 배열 선언 방법]

        1. const arr = new Array(); // 0 칸짜리 비여있는 배열 생성
        2. const arr = new Array(3); // 3칸짜리, 모든칸의 값은 비여있는(empty) 상태
        3. const arr = []; // 0칸짜리 배열 생성
                            // []: 배열기호
        4. const arr = [값1, 값2, 값3] // 3칸짜리, 값이 차있는 배열 생성

        index : 각 자리 위치
        length: 총 길이
