정규표현식(RegularExpression , RegExp)

        - 정규 표현식 또는 정규식
        - 특정한 규칙을 가진 문자열 집합을 표현하는데 사용하는 형식 언어
        - 입력된 문자열에 대한 특정 조건 검색, 일치여부 판단, 치환에 대한 
            조건문을 간단히 처리할 수 있음

정규 표현식 객체생성 및 확인하기

        1. 정규표현식 객체 생성 방법 

            1) const regExp = new RegExp("정규표현식")
            2) const regExp =/정규표현식/;

        2. 문자열 패턴 확인 기능(메서드 = 함수)

            1)regExp.test(문자열)

정규표현식으로 표현하기

        문자열의 패턴을 나타내는 문자
        문자마다 지정된 특별한 뜻이 담겨있다

        일반 문자열 : 같은 문자열이 존재하는지 검색
        -> /abc/ abc 라는 문자열이 존재하는지 검색


        --------------------------------------------------------

        [] : 대괄호 안의 어떤 문자와 일치
        ->/[abcd]/   a, b, c, d 중 하나라도 일치하는 문자가 있는지 검색

        [0~9] : 0부터 9까지 모든 숫자 사용가능
        [a-z]
        [A-Z]
        [ㄱ-횧] : ㄱ부터 힣까지 모든한글
        [가-힇] : ㄱ부터 힣까지 모든한글(모음,자음 제외)

        --------------------------------------------------------

        [^] 대괄호 안의 문자와 일치하지 않음
        ->[^abcd] a,b,c,d 중 하나라도 일치하는 문자가 없음

        ^(캐럿) : 문자열의 시작을 의미
        /^a/  a 로 시작한다는 의미

        $(달러) : 문자열의 끝을 의미
        /a$/  a 로 끝

        --------------------------------------------------------
        escape 문자
        - 특정 문자를 특별한 용도로 사용하려는 경우에 사용되는 문자가

        1) \ + 특수문자 : 기능이나 패턴이 아닌 문자 그대로 인식

        2) \d(digit, 숫자) : 아무숫자 0~9중 하나
            \w(word, 단어) : 아무문자(띄어쓰니, 특수문자, 한글불가)
            \s(space, 공백): 아무 공백 문자

        --------------------------------------------------------
        * 수량 관련 문자
        a{5} : a 문자가 5개 존재하는가
        a{2,5} : a가 2개이상
