자바스크립트의 변수(variable)

        메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간
        변수에 이름을 붙이고 원하는 값을 대입하면
        필요한곳에서 변수명을 호출하여 저장된 값을 사용할 수 있다
        
변수이름(변수명)명명규칙

        -의미 있는 이름을 작성
        대, 소문자 구분
        특수문자는 _ 사용가능(첫글자 제외)
        숫자사용가능(첫글자 제외)
        카멜표기법 작성 - 문장 여러단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성 변수명. 작성시 띄어쓰기가 안됨
        curioustore :변수명 짓기 사이트

자바스크립트 변수의 종류와 특징

        변수이름을 새로 만들어서 변수가 생기는 것을 담을 공간 생성
        새로 만들었다를 어떻게 표기할 것인가?

        new 바구니 a = '물건이나 숫자가 담겼습니다';
        var 바구니 a = '물건이나 숫자가 담겼습니다';
        var : 변수, 변수명 중복가능(덮어쓰기 가능)
        당연히 문제가 발생
        let : 변수, 중복안됨

        const: 상수 변수명 중복, 값변경도 안됨
        (최신 트렌드, 제일 많이 사용)

        1순위 : const(js는 특정 요소를 선택해서 사용하는 경우가 많아 변수에 고정시켜둠)
        2순위 : let
        3순위 :

자바스크립트의 자료형

        자바스크립트는 변수 선언시 별도의 자료형을 지정하지 않음
        -> 변수에 대입되는 값(리터럴)에 의해 자료형이 결정됨.

        - 숫자는 '' "" 없이 표기 가능
        - 문자열 '' "" 모든 언어에 필요(엉어포함)
        - 진실 혹은 거짓 true / false 는 ''""없이 표기가능
        - function
        - undefinded (정의되지 않은 변수) -> 선언만 되었고, 값이 대입되지 않음
