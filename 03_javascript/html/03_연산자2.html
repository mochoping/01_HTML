복합 대입 연산자(+=, -=, *=, /=, %=)

        - 기존 변수에 저장되어있는 값에
        새롭게 대입되는 값의 산술 연산 결과 대입

        (증감 연산자는 1씩 증감, 복합대입 연산자는 원하는 만큼 증감)

        예시
        let temp = 10;
        temp += 5
        결과 : temp 는 15로 변경

현재 값 : 0

+=(더하기 연산자)

        기존 값에 새로운 값을 더하거 이어붙여 업데이트 하는 연산자는
        == 타입 알아서 수정

        += 글자글자

        자동속성변화는 대부분 글자 우선. 원한다면 넘버로 재조정 가능
