JavaScript 란?

        웹 브라우저 에서 주로사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어
        ECMA script표준을 따르는 대표적인 웹 기술

JavaScript 코드 작성 방법

  1. 브라우저 개발자도구 -> console에 직접 작성
  2. HTML 내부에 script 태그를 이용하여 작성 body 나 head 등의 html 태그 내부에서 작성
  3. HTML 외부에 존재하는 .Js파일을 연결해서 사용하는방법,
    .css와 비슷하게 사용 css 와 다르게 파일을 불러옴
  4. 태그에 직접 js 코드를 직접 작성하는 방식(권장되지않음)
    css 스타일 직접 꾸미기 방식
