분기문

        -반복문 수행중 반복을 멈추거나 넘어가는 구문

        -break : (가장 가까운 / 감싸고 있는) 반복문을 멈춤

        -continue : (가장 가까운 / 감싸고 있는) 반복문의 시작 부분으로 
                    -> 다음 반복으로 넘어감

중첩 반복문(2중 for문)

    -반복문 내부에 또 반복문이 작성된 형태

    [작성법]
    for(초기식1; 조건식1; 증감식1){
        // 내부에 있는 for문은 초기식 1이 true 일때 실행
        for(초기식2; 조건식2; 증감식2){
            초기식1과 초기식 2가 true 일때 실행
        }
    }

    메모리 소모량이 많아 가급적 사용 자제. 다른좋은방법이 있다고 함

while 문

    - 끝이 정확하게 정해져 있지 않은 반복에서 주로 사용
    (ex: 어떤 값이 입력되기 전 까지 반복)

    [작성법]

    while(조건식){
        조건식이 true 일 경우 수행할 코드;
    }
    - 조건식이 true 인 경우 내부 코드를 1회 반복 후 다시 조건식 검사 진행
    -> false 가 나올 때 까지 무한 반복
