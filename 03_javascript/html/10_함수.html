return (반환)

        - 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감

        - 호출한 곳으로 돌아갈때,
            return 옆에 작성된 값을 가지고 돌아갈 수 있다,

        return 100; = 호출한곳으로 100을 가지고 돌아갈 수 있음.

무한한 결과가 아닌, 선택지가 있는 결과를 작성할때 매개변수와 전달인자를 사용 매개변수와 전달인자는 한 세트, 매개변수는 전달인자를 전달해주는 전달 역활뿐 전달인자는 위에서 선택된 전달인자가 들어옴 매개변수 전달인자 function 결정사( let 소개팅대상 ="홍길동"; ){ }

익명함수

      [기본함수]
      function 함수명([매개변수]){
        //함수정의
      }

      함수명(); // 이름이 있으니까 함수에서 작성한 함수명으로 어디서든지 호출 가능

      *********************************
      [익명함수]
      function([매개변수]) {
        //함수정의
      }
      이름이 따로 부여되지 않아 이후 재호출이 불가능.
      *기본 함수와 다르게 변수에 대입, 매개변수로 바로 전달되어야 하는 곳에서 주로 사용

화살표 함수(arrow function)

        -익명 함수의 한 종류
        - 익명 함수를 간단히 표기하는 방법
        익명 함수 : function(){}

        1. 기본 형태 : () => {}

        
        2. 매개변수 1개 :(e) => {}
                         e => {} //()생략가능
        3. 매개변수 n개 : (x,y,z) => {} () 생략할 수 없음
