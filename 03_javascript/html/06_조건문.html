조건문 (condition)

        condition = 어떤 조건에 따라 다르게 동작하는 것을 의미
                    어떤 상태에 따라 다르게 동작하는 것을 의미
                    조건      = 만약 이 조건이 참이라면 이 작업을 수행
                    상태 상황 = 특정한 상태나 상황에서만 이 작업을 수행

        프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로 
        작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if(만약에...)

        
            if(조건식) {
                조건식이 true 인 경우 실행할 코드;

            }
        

        * 조건식 : 결과가 true, false 인 식(연산)

        * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문 (if. switch)

입력받은 수가 양수인지 아닌지 검사

if(만약에) - else(아니면) 문

      
        if(조건식) {
          조건식이 true 인 경우 실행할 코드;
        } 
        else {
          if에 작성한 조건식이 false 일 경우 실행할 코드
        }
        * else항목에는 조건식을 작성하지 않음

        삼항연산자
        if 조건식          true      else      false
        (             ) ?             :               ;
    
      
      if - else 문 vs 삼항연산자
      if - else 문 : true나 false 일 때 수행해야하는 구문이 많을 때 많이 사용
      삼항연산자    : 구문이 적을때 사용

if(만약에) - else if - else

      
        if(조건식a) {
          조건식이 true 인 경우 실행;
        } else if (조건식b){
          조건식a false 이면서 b가 true 인 경우
        } else {
          모든 조건식이 false 인 경우 실행
        }
      

      *else if 구문은 if 와 else 사이에 원하는 만큼 작성 가능
       대신 무조건 if와 else 사이에 작성해야함

switch문

switch, case, default문

      switch 문
      - 식 하나의 결과로 많은 경우의 수를 처리할 때 사용하는 조건문

      
          switch(식) {//식의 결과는 정수 또는 문자열
            case 결과1:
              결과1일 때 수행 코드;
              break; 결과 1일때 멈추기
            case 결과2:
              결과2일 때 수행 코드;
              break; 결과 2일때 멈추기
            case 결과3:
              결과3일 때 수행 코드;
              break; 결과 3일때 멈추기
            ...
            default:
              위 모든 케이스에 만족하지 못할 때 수행할 코드
              (if문의 else와 같음)
          }
