transition 과 transform

transition 속성

    transition : 요소의 속성이 변경될 때 그 변화가 즉각적으로 나타내는 대신, 
                부드럽게 전환되도록 애니메이션 효과를 추가
                transition : 속성명 전환시간 전환속도곡선 시작전지연시간
            예시 :
            배경을 빨간색으로
            3초후에 변경이 시작됬으면 좋겠고
            변화는 0.5 초 속도로 느리게 변황되었으면 좋겠음
            일정한 속도로 변환

            transition: backgound-color 0.5 linear 3s;
            속도와 몇초후 시작하라는 표기는 자주 사용 안함
            (hover 나 active 같은 효과에서 빨간색을 지정해주어야함. 자체적으로는 지정 하지 못함)
        
            트랜지션은 배경색 글자크기 글자섹 사로세로크기등 다양한 스타일 속서ㅇ값에
            속도변화를 줄 수 있음

            트랜스폼은 이동하기 기울이기

            상대적으로 트랜지션이 트랜스폼보다 많이 사용

transform 속성

    transform : 요소를 이동, 회전, 크기조절, 뒤집기등의 변환을 적용하는데 사용
    1. translate : 요소를 x, y축 방향으로 이동
        transform:translate(10px,20px); 가로로 10 새로로 20 픽셀 이동 
                            숫자 + 3글자 오른쪽과 아래쪽 - 는 반대
        transform: rotate(45deg)       → 시계방향             
        transform: scale(1.5)  태그와 태그안에 있는 값 = 요소의 크기를 1.5배 키움
        transform: skew(10deg, 5deg)  가로 오른쪽으로 10도, 세로 오른쪽으로 5도 기울이기
        
        예를 들어 요소를 오른쪽으로 20px 아래로 10px 이동하고 크기를 1.2배로 키운다.
        기능이 2개이상일때 작성법

        transform:translate(20px,10px)
