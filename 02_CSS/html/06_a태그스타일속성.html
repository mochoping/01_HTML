하이퍼링크관련 태그 - a 태그

        하이퍼링크태그
        a태그 내부에 href 속성을 작성하면 글자 색상과 밑줄이 생김
        파란색 - 아직 클릭하지 않은 링크
        보라색 - 방문한 링크 색상

        a 태그는 사용자가 특정 행동을 했을 경우 3가지 경우로 나누어 설정

        a:hover = 마우스링크 위에 올렸을 때 스타일 설정
        주로 text-dcoration : underline
        a:active = 마우스를 클릭했을 때 활성화 되는 스타일
                주로 글자색 또 한 번 변경하고 클릭시 약간 줄어들거나 커지게 만들어줌
        a:visited = 사용자가 이미 방문한 링크에 스타일을 지정할 때 사용
            주로 color만 사용
        button, input type="submit" 과 a 태그의 차이점
            제출을 위한 용도로 주로 사용되고, 
            a href 는 이동하기 위해 사용
