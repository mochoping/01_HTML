position

        요소 배치(위치) 관련 스타일
        위치를 지정하는 속성
       - relative(상대적인)
            지정된 요소 내부에 다른요소가 상대적인 위치를 지정할 수 있도록 
            기준이 되게 만드는 속성
       - absolute(절대적인)
            기본 요소 배치 순서를 무시하고 지정된 절대위치에 요소를 배치 
            z-index : 정수 가장 우선으로 보여지는 디자인

            z-index : 정수
            가장 우선으로 보여지는 디자인 순서를 지정
            숫자가 클수록 먼져보임
            단독으로도 쓸때가 있음
            최대 최소는 지정된 숫자 없음
            32비트 정수형

       - fixed (고정된)
            항상 고정된 위치에 요소를 배치함
            (화면 움직임과 무관)
       - stiky
            원래 요소가 보여지는 화면이라면 position X
            요소가 보여지는 화면을 벗어나면 fixed(고정)

position:fixed
