화면 배치(표시,형식) 지정 :display 속성

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성

        -block : 화면을 수직 분할(행을 나눔, 한줄 다 차지)
            +width, height 지정 가능

        -inline : 화면 수평 분할(하나의 행의 열을 나눔 == 글자, 문자열 생각)

        -inline-block : inline의 수평 분할 + blockdml width,height 지정

        -none : 요소는 존재하나 화면에는 보이지 않게 함

        -flex : 요소의 정렬되는 방향, 요소간의 간격을 유연하게 처리하는 방식

        -grid : 격자무느 형식으로 화면을 쉽게 분할하는 방식

block 형식의 요소를 inline 형식으로 변경
