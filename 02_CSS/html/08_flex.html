display : block 으로 되어있는 형식들 inline 형식으로 가로로 모두 정렬
        justify-content = 가로로 정렬시킨 태그들을 자동너비주기를 실행해서 너비를 일정하게 유지
        text-align : 글자정렬. 가로 가운대 정렬
        align-item : 글자로 작성된 문장들을 하나의 아이템으로 보는것
                    세로에서 가운데 정렬할때 주로 사용

        text    -    align    -    items
        가로로                     세로로

flex 레이아웃

        display : flex;
        div 안에 작성한 태그들을 유연하게 배치 할 수 있는 속성
        제일많이 쓰이는 기능
        수평이나 수직으로 정렬이 가능
        flex의 경우 div(=가로 한줄 전체 차지하는 Block)형태를 inline 형태로 바꾸는 효과가 있음

        display : flex; 와 같이 자주 쓰이는 속성 존재
        1.  가로로 존재하는 태그들의 여백을 설정해주는 방법
        1-1. justify-content:center;
            flex 컨테이너 내의 자식 태그를 가로방향으로 가운데 정렬
        1-2. justify-content:space-around;
            flex 컨테이너 내에서 자식 태그들 사이에 균등한 여백을 주어 배치

        2.  세로로 존재하는 태그 줄의 여백을 가운대로 설정해주는 방법
            align-item : center;
            세로 줄에서 가운데 정렬
            주로 flex-direction:row; 
            와 함께 사용할 때 세로 가운데 정렬이 필요할 경우 사용
            text - align - items
            └  가로  ┴  세로  ┘
        3.  한 줄에서 있을 수 있는 태그의 너비가 넘어가면 자동으로 줄바꿈이 되도록 설정되어있음
            flex-wrap : wrap;
            최대한 가로에서 세로로 줄바꿈이 되지 않도록 방지, 넘어가면 넘어가짐

flex 와 inline 차이

특징 요소 배치 방식 크기 제어 정렬 옵션 응용 범위
inline 같은 줄에 나열되며 필요한 공간만 차지 높이, 너비 지정 어려움 정렬 옵션 제한적 텍스트 및 링크 요소 등에 사용
flex 가로, 세로 방향으로 유연하게 배치 크기, 여백 쉽게 제어 가능 다양한 정렬 옵션 지원 전체 레이아웃 설계 주로 사용
