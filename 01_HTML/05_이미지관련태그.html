img 태그 사용하기

    사진이나 그림같은 이미지를 삽입하는 용도
    1. src : 삽입할 이미지의 경로를 작성하는 속성 (위치, 주소)
    2. width / height : 너비 높이 지정
    3. alt : 이미지에 대한 설명을 작성하는 속성
        -> 이미지가 보이지 않을경우 화면에 이미지 이름이 출력
        -  스크린 리더가 이미지 설명을 읽어줌
        *  스크린 리더란? 화면의 텍스트를 읽어주는 프로그램

    이미지는 닫는 태그 없이 단독으로 한번만 태그 작성
