확장프로그램 종류

    1. auto rename tag : 태그명이 변경될 때 닫는 태그도 자동으로 변경 
    2. code spell checker : 코드에서 철자나 문법이 문제가되어 에러가 생기지 않도록 도와줌 
    3. highlight matching tag : 태그가 제대로 닫아졌는지 순서는 맞는지 확인해줌 
    4. image preview : 코드내에서 이미지 위에 마우스 오버시 미리보기 가능 
    5. korean language pack : 한국어로 vscode 설정 변경 
    6. live preview : html 에 작성한 코드와 디자인을 미리보는 확장 프로그램 
    7. prettier - code formatter : 코드정렬을 개발자 형식에 맞게 정렬해주는 확장 프로그램
